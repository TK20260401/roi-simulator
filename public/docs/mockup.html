Client-Side 計算（APIなし）
入力値が変わるたびに useMemo で即時再計算

lib/calc.ts — calculate(inputs)

operatorCost
 = operatorCount × personMonthCost
 = 25 × 80 = 2,000万（自動算出）

currentMonthlyCost
 = operatorCost + systemCost + trainingCost/12 + otherCost
 = 2000 + 300 + 500/12 + 100 = 2,441.7万

reducedOperatorCost
 = max(0, 2000 - 12 × 80) = 1,040万

afterMonthlyCost
 = 1040 + 200 + 10(API) + 教育削減 + 100

ROI = (年間削減 - (AI+API)年額) / 総投資 × 100

paybackMonths = 初期投資 / monthlySaving
