Client-Side (ブラウザ内)
lib/calc.ts — calculate()
入力 → useMemo → 即時計算
0. operatorCost = operatorCount × personMonthCost（自動算出）
1. currentMonthlyCost = 人件費 + システム + 教育/12 + その他
2. reducedOperator = max(0, 人件費 - 削減人数 × 人月単価)
3. afterMonthlyCost = reduced + AI費 + API費 + 教育削減 + その他
4. monthlySaving = current - after
5. ROI = (年間削減 - (AI+API)年額) / 総投資 × 100
6. payback = 初期投資 / monthlySaving
保存時のみAPI
POST /api/simulations
15個の入力値 + name + company_name + display_name を送信
